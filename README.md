# Ekstraklasa HUB

A web application dedicated to the Polish top-tier football league (Ekstraklasa), designed to provide statistics, match data, or team information.

## 📋 Project Description

Ekstraklasa HUB is a platform that aggregates and presents data related to football matches. The application features a Python-based backend handling the logic and data processing, coupled with a custom frontend designed with HTML and CSS. It serves as a central hub for browsing league-related information.

## 🚀 Features

* **League Data:** Access to information about Ekstraklasa teams and matches.
* **Data Visualization:** Clear presentation of statistics and standings.
* **User Interface:** Responsive and clean interface built with custom CSS.
* **Backend Logic:** Robust data handling powered by Python.

## 🛠️ Tech Stack

The project relies on the following technologies:

* **Python 3.x** - Core backend logic.
* **Django** - for handling HTTP requests and routing.
* **HTML5** - Structure of the web pages.
* **CSS3** - Styling and visual layout.

## ⚙️ Prerequisites

To run this application locally, ensure you have the following installed:

* Python 3.8 or higher.
* pip (Python package installer).

## ▶️ How to Run

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/suwinho/ekstraklasaHUB.git](https://github.com/suwinho/ekstraklasaHUB.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd ekstraklasaHUB
    ```

3.  **Create and activate a virtual environment (optional but recommended):**
    ```bash
    python -m venv venv
    # Windows
    venv\Scripts\activate
    # macOS/Linux
    source venv/bin/activate
    ```

4.  **Install dependencies:**
    If a `requirements.txt` file is present:
    ```bash
    pip install -r requirements.txt
    ```

5.  **Run the application:**
    *If using Django:*
    ```bash
    python manage.py runserver
    ```
    *If using Flask:*
    ```bash
    python app.py
    ```

6.  **Access the app:**
    Open your browser and visit: `http://127.0.0.1:8000`.
